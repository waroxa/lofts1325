User-submitted reviews for your Hotel Booking plugin accommodations.

== Description ==

User-submitted reviews for your Hotel Booking plugin accommodations: display ratings, easily manage reviews via native WordPress comments.

== Installation ==

1. Upload plugin to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
